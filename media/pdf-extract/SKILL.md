# PDF Extract

## Quand l’utiliser
- extraction texte/tableaux depuis PDF
- remplissage de formulaires PDF
- fusion/séparation de PDF

## Entrées
- fichiers PDF fournis par l’utilisateur

## Sorties
- texte extrait
- tableaux structurés
- PDF modifié si applicable
